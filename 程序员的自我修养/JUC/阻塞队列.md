 ## 四组API

| 方式                 | 添加                    | 移除                | 检测队首元素 |
| -------------------- | ----------------------- | ------------------- | ------------ |
| 抛出异常             | add                     | remove              | element      |
| 有返回值，不抛出异常 | offer()                 | poll()              | peek         |
| 阻塞等待             | put()                   | take()              | -            |
| 超时等待             | offer(e, timeout, unit) | poll(timeout, unit) | -            |
